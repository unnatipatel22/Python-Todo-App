# 📝 Python Todo App

A simple and efficient command-line Todo List application developed using Python. This project helps users organize and manage daily tasks by allowing them to add, view, update, and delete tasks. It was created to strengthen Python fundamentals while practicing file handling, functions, loops, and user interaction.

---

## 📌 Features

- Add new tasks.
- View all saved tasks.
- Update existing tasks.
- Delete completed tasks.
- Simple command-line interface.
- Beginner-friendly and easy-to-understand code.

---

## 🛠️ Technologies Used

- Python 3

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/unnatipatel22/Python-Todo-App.git
```

Move into the project directory:

```bash
cd Python-Todo-App
```

Run the application:

```bash
python todo_app.py
```

---

## 💡 How It Works

1. Run the Python program.
2. Select an option from the menu.
3. Add, view, update, or delete tasks.
4. The application keeps track of your task list.
5. Continue managing tasks until you choose to exit.
